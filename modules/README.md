# Modules

Hands-on exercises for each module of the [Product Manager Learning Path](https://malston.github.io/claude-code-wiki/training/product-manager-path/).

| Module                   | Directory                | Focus                                |
| ------------------------ | ------------------------ | ------------------------------------ |
| 1. How Claude Code Works | `01-claude-code-basics/` | Setting expectations                 |
| 2. Design Principles     | `02-design-principles/`  | Recognizing good design              |
| 3. Architecture          | `03-architecture/`       | Evaluating trade-offs                |
| 4. Coding Standards      | `04-coding-standards/`   | Why conventions matter               |
| 5. TDD                   | `05-tdd/`                | Writing testable acceptance criteria |
| 6. Workflows             | `06-workflows/`          | PR review, feature decomposition     |

Each directory has a `README.md` with exercise instructions. Work through them in module order -- later modules build on earlier ones.
